CSS: Color

Element Description Arguments Example (Input) Example (output)
color: Styles the color of an element (foreground). Name, hexadecimal, rgb(a), or hsl(a) color: blue; Example
background-color: Styles the background color of an element (background). Name, hexadecimal, rgb(a), or hsl(a) background-color: lavenderblush; Example
color:#000000; Hexadecimal code, reprsentative of color. 6 digit string: 0-9, A-F. color: #9F0ECB; Example
color: rgb( x, y, z); Red, Blue, Green code, representative of color. x,y & z are integers in [0, 255]. color: rbg( 255, 0, 120); Example
color: hsl( x, y, z); Hue, Saturation, Lightness code, representative of color. x is an integer in [0,360]. y and z are percentages in [0%, 100%]. color: hsl( 209, 92%, 33%); Example
color: rgba( x, y, z, a); Red, Green, Blue, Alpha code. Similar to rgb, but includes a for opacity. x, y & z are integers in [0, 360]. a is in [0,1]. color: rgba( 255, 252, 0, 0.5); Example
color: hsla( x, y, z, a); Hue, Saturation, Lightness, Alpha code. Similar to hsl, but includes a for opacity. x is an integer in [0,360], y and z are percentages in [0%, 100%], and a is in [0,1]. color: hsla( 258, 77, 50, 0.5); Example
color: transparent; Gives an element an opacity of 0, rendering it transparent. N/A color: transparent; Example

CSS: Typography

Element Description Arguments Example (Input) Example (output)
font-family: Selects a font. 'Times New Roman', Helvetica, Candara, etc. font-family: Garamond; Example
font-family: x, y, z; Selects a font, x, with back-up font y and additional back-up font type z. x and y are font names, z is either serif or sans-serif. font-family: 'Times New Roman', 'Georgia', serif; Example
font-weight; Controls how thin or thick text appears. bold, normal, lighter, bolder, or an interger value in [1,1000]. font-weight: 800; Example
font-style: italic; Italicizes text. N/A font-style: italic; Example
text-transform: Transforms text to either all uppercase or all lowercase. uppercase or lowercase. text-transform: uppercase; Example
letter-spacing: Changes the space between letter of text. px or em (em is recommended). letter-spacing: 1.3em; Example
word-spacing: Changes the space between words of text. px or em (em is recommended). word-spacing: 30px; Example Example
line-height: Changes height of the line containing text. px, em, %, or a unitless value (unitless value is recommended, as it multiplies the default). line-height: 1.5; Example
text-align: Aligns text. Left (default), right, center, or justify. text-align: center; Example
